Positions

  • static (default) - does nothing
  • relative
  • absolute
  • fixed
  • sticky
  • top, right, bottom, left, z-index (not positions)

The element with position: relative | absolute | fixed | sticky, gains access to top, right, bottom, left and z-index styles.

z-index controls elements layers. Can only be used on elements with positions of relative | absolute | fixed | sticky AND displays of flex | grid.

Relative

Top, right, bottom and left styles move element away from it's original position by the given amount of pixels.

Element with position: relative, sets the boundaries for it's children element with position: absolute.

Absolute

Element loses it's place on the document flow. Top, right, bottom and left styles move element away from it's closest ancestor element with relative position.

Fixed

Element loses it's place on the document flow. Top, right, bottom and left styles move element away from current viewport. Fixates element on specific position on the screen and it does not scroll up or down, but always stays in that place while we scroll the page.

Sticky

Element works similarly to fixed, but does not lose it's place on the document flow. Top, right, bottom, left sticks element to the parent element borders once it reaches the sticky element.

Icons

You can add icons to your webpage from different icon libraries. First step - select library you want: bootstrap, google icons, font-awesome, mui, tailwind, ant-d... Second step - import or link CDN. Third step - use icon. While styling icons think of them as simple letters. Change their font-size, not width and height and so on.

Attribute Selectors

  • [attribute] - attribute name
  • [attribute='value'] - attribute name and it's value
  • [attribute*='value'] - attrbite name and it's partial value
  • [attribute~='value'] - attribute name and it's one of the values
  • [attribute|='value'] - attribute name and it's value or it's value'-'
  • [attribute^='value'] - attribute name and it's value start
  • [attribute$='value'] - attribute name and it's value end
