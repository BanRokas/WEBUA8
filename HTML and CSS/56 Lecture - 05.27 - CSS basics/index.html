CSS Basics

CSS - Cascading Style Sheets used to style HTML elements.

Ways to write CSS

  • Inline - CSS is written inside of a tag using style attribute.
  • Internal - CSS is written inside of HTML file, inside of style tag.
  • External - CSS is written in CSS file that is being linked to HTML file.

CSS Selectors

CSS selectors dictate which styles are given to which elements.

  • tag - p/h1/span/div... - you select all tags in HTML you wrote as selector.
  • * - everything - selects every element.
  • .className - selects elements that have specific class.
  • #idName - selects elements that have specific id.
  • ...

Colors

  • named colors (only ~140)
  • hex - hexadecimal
  • rgb - red green blue
  • rgba - red green blue alpha (alpha stands for opacity 0-1)
  • hsl - hue saturation lightness

Displays

  • Inline - elements can be placed in the same row and only takes the space needed to display the element. Can't set height or width styles.
  • Block - elements will be placed one under the other and will take the entire row. Can set height and width styles.
  • Inline-Block - elements can be placed in the same row and take the space needed to display the element. Can set height and width styles.
