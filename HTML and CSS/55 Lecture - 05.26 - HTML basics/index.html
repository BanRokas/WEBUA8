      Keyboard shortcuts

      • CTRL A - selects everything
      • CTRL ENTER - creates new line (not line break)
      • ALT TAB - jump in between the programs
      • CTRL Z - cancels previous actions (undo)
      • CTRL Y - opposite of CTRL Z (redo)
      • CTRL F - opens search
      • CTRL D - multi selects selected content duplicates
      • ALT mouseClicks - creates multiple cursors
      • SHIFT ALT F - automatically tabulates the file
      • CTRL / - comments out selected lines
      • CTRL T - creates new tab (for browser)
      • ALT Z - toggles word wrap
      • SHIFT ALT DOWN/UP - duplicates selected lines up or down
