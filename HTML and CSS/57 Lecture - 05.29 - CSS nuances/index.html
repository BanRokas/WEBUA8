Browser default styles

Most tags have their own styles given by the browser. Different browsers sometimes have different styles.

Normalize CSS - normalizes styles in betweem different browsers.

Reset CSS - removes the default styles.

Parent and children elements

Element CAN have many children elements.

Element HAVE only one parent element.

Inheritance

Styles can be inherited by children elements from their parent elements.

CSS styles priorities

  1. !important
  2. Inline
  3. Selector Specificity
    1. 100 - id
    2. 010 - class
    3. 001 - tag
  4. Order (only if specificity is the same)
  5. User agent stylesheet / browser styles
  6. Inheritance

Box model

  1. Content - center part of an element
  2. Padding - in between Content and Border
  3. Border - in between Padding and Margin
  4. Margin - in between our elements and the elements around it
  5. Positions - talk about it next lecture
