Flex

We use display of Flex then we need to display elements on the screen with a bit more control over them as a group. Even though we do have great control over the group of those elements, we don't have great control over every single separate element (grid is better for that).
Flex has quite a few properties, they are separated in two groups: parent (flex container) and children (flex items).

Flex properties

Parent

  • display: flex;
  • flex-direction: row (default) | column | row-revese | column-reverse;
  • flex-wrap: no-wrap (default) | wrap | wrap-reverse;
  • flex-flow: flex-direction flex-wrap;
  • justify-content - how elements are positioned on the main axis
  • align-items - how elements are positioned on the cross axis (for each separate row)
  • align-content - how elements are positioned on the cross axis (for all the items (overrides align-items style))
  • gap: row-gap column-gap;

Children

  • order - change order of elements
  • flex-grow
  • flex-shrink
  • flex-basis
  • flex: flex-grow flex-shrink flex-basis
  • align-self
