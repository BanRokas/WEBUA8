Responsive Design

It is used to create different layouts depending on user's screen size.

Breakpoints

The breakpoints - the specific amounts of pixels your screen changes the layouts of elements once screen reaches specific size.
There are no golden standards for breakpoints. You should choose the breakpoints that looks nice for your personal website application. But if u do need some guidlines - choose either bootstrap or tailwind suggested breakpoints.

Mobile first

Mobile first design principle - is that we're styling our website for mobile applications first and once we're done with that we move on to the bigger screen device styles and so on and so on...

That is done because majority of website users are using mobile devices so we're focusing on majority of users. It is easier to add more content once screen expands rather than remove some content then screen shrinks...

Using Mobile-First design principal we need to write css using media queries with (min-width) instead of (max-width).
