Frameworks / libraries

Library/Framework - someone's else written code you can use in your own project.
To use a library you simply need to import it's CDN to your own project and use it's classes.

CSS Libraries

  • Bootstrap
  • Tailwind
  • Bulma
  • UI Kit
  • Materialize
  • ...

Pros

  • Makes styling HTML way faster
  • Reduces the need to write CSS
  • Integrated responsiveness
  • Helps to avoid writing redundant code
  • ...

Cons

  • You forget how to write base CSS
  • Libraries can only do what they're created to do
  • Poluting HTMl code with bunch of classes everywhere
  • Can get into selectors specificity problems while trying to apply your own css to HTML elements
  • You have to spend time learning the library
  • Not unique design of websites using same library
  • ...

Preprocessors

CSS with superpowers

Preprocessors allows you to write CSS with additional tools. You can easily create variables, nest CSS code, write conditions, loops, functions, mix'ins and so on...

Whenever we write code using preprocessors, we need to convert (compile) that code into base CSS code before we link it to HTML.

List of preprocessors

  • SASS/SCSS
  • LESS
  • Stylus
  • PostCSS
  • ...
