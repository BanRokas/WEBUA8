Transform

Transform - style that allows us to change the appearance of the element in ways described below:

  • rotate() - rotating element
  • scale() - changing it's size
  • translate() - moving element around
  • skew()
  • matrix()
Transform example

transform-origin

transform-origin tells the element from which position it has to do the transformation (default is center)

Transition

Transition is controlling how elements goes from one state to another.
Most commonly transitions are being used together with :hover pseudo class.
Transition only happens once per action. From default to change dand back after you remove the cursor or whatever the action was that triggered the transition.

  • transition-duration - how long transition takes
  • transition-timing-function - how time is used to display the whole transition
  • transition-delay - how long tansition is delayed
  • transition-style - for which style stransition options are assigned
  • transition-behavior
  • transition - combination of all transition styles

Animation

Animation is controlling how element changes in between many states.
For animation to start, there is no need for user interaction, although there can be.
Animations can repeat as many times as we want and during those animations, many different styles can change multiple times.

  • animation-duration
  • animation-delay
  • animation-timing-function
  • animation-name
  • animation-iteration-count
  • animation-direction
  • animation-play-state: running | paused;
  • animation-fill-mode
  • animation-composition
  • animation: combination of all animation styles

Keyframes

To describe how animation is happening, we write keyframes describing how the element is styled at a specific time in the animation.
(example: @keyframes animationName: { 0%{} 50%{} 100%{} })
