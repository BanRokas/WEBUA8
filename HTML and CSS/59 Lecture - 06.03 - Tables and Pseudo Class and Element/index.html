Table

Used to display data as a table with headings in thead and data in tbody.
In HTML tables are created by creating rows. Columns in those rows are created by filling the rows with data or headings.

CSS nesting

We can nest our CSS code, by writing one selector inside of the other selector. This way we write less selectors saving code space and time.

Pseudo Class selectors

Some pseudo classes reacts to the state of the element. Others to the position of the element and so on.

  • Links
    1. :link - unvisited new link
    2. :visited - visited link
    3. :hover - mouse over the element (not just links)
    4. :active - reacts to the click (moment of clicking)
  • :first-child
  • :last-child
  • :only-child
  • :first-of-type
  • :last-of-type
  • :only-of-type
  • :nth-child(n)
  • :nth-last-child(n)
  • :nth-of-type(n)
  • :nth-last-of-type(n)
  • No need to know
    • :root
    • :not(selector)
    • :empty
    • :fullscreen
    • :lang(language)
    • :target
  • ...

Pseudo element selectors

  • ::after - creates content after selected element
